## 🔍 نکات تست مهم

### 1. تست فیلدهای اجباری vs اختیاری
- فیلدهای اجباری: دانشگاه، شماره دانشجویی، رشته، مقطع
- فیلدهای اختیاری: بقیه موارد

### 2. تست محاسبات مالی
- سهم عامل و مدیر باید خودکار محاسبه شود
- مجموع سهم‌ها = مبلغ کل

### 3. تست فیلترها و جستجو
- جستجو در نام دانشجو
- فیلتر بر اساس وضعیت
- فیلتر بر اساس تاریخ

### 4. تست رابط کاربری فارسی
- تمام متن‌ها باید فارسی باشند
- تاریخ‌ها باید شمسی نمایش داده شوند
- راست‌چین بودن متن‌ها

## 🐛 مشکلات احتمالی و راه‌حل

### اگر پنل ادمین باز نمی‌شود:
```bash
python manage.py runserver
# سپس به http://127.0.0.1:8000/admin/ بروید
```

### اگر داده‌های نمونه وجود ندارد:
```bash
python manage.py create_sample_data
```

### اگر migration خطا می‌دهد:
```bash
python manage.py makemigrations
python manage.py migrate
```
