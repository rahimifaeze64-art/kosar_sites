## مشکلات رایج و راه‌حل

### مشکل 1: CORS Error
**علامت:** خطای CORS در Console مرورگر

**راه‌حل:**
- مطمئن شوید فرانت روی `localhost:8080` اجراست
- بک‌اند روی `127.0.0.1:8000` باشد

### مشکل 2: 403 Forbidden
**علامت:** خطای 403 در درخواست‌های POST

**راه‌حل:**
- ابتدا تست اتصال را اجرا کنید تا CSRF Token دریافت شود
- سپس Login کنید

### مشکل 3: سرور بک‌اند متوقف شده
**راه‌حل:**
```bash
cd backend
python manage.py runserver
```
