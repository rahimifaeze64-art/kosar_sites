#### 2. فعال‌سازی API در فرانت

## 🚀 دستورالعمل راه‌اندازی

### 1. نصب و راه‌اندازی بک‌اند

```bash
# نصب وابستگی‌ها
cd backend
pip install -r requirements.txt

# اجرای migrations
python manage.py migrate

# ایجاد داده‌های اولیه
python create_initial_data.py

# راه‌اندازی سرور
python manage.py runserver
```

**اطلاعات ورود پیش‌فرض:**
- مدیر: `manager` / `123456`
- کارمند: `zahra` / `123456`
- عامل: `masoumi` / `123456`
- دانشجو: `qasim` / `123456`

### 2. راه‌اندازی فرانت‌اند

```bash
# نیاز به یک وب سرور ساده
# گزینه 1: Python
python -m http.server 8080

# گزینه 2: Node.js
npx http-server -p 8080

# گزینه 3: PHP
php -S localhost:8080
```

سپس مرورگر را باز کنید:
```
http://localhost:8080/index.html
```

### 3. تست اتصال

```
http://localhost:8080/test-connection.html
```
