# راهنمای پیاده‌سازی ساید بار

## تغییرات انجام شده

### 1. ساید بار سمت راست
- ساید بار حرفه‌ای با طراحی مطابق تصویر ارسالی
- قابلیت باز و بسته شدن در موبایل
- همیشه باز در دسکتاپ (صفحات بزرگتر از 1024px)
- گرادینت آبی نیلی زیبا
- انیمیشن‌های نرم و روان

### 2. محتوای ساید بار
**بخش بالا:**
- لوگو و عنوان سیستم
- جستجوی سریع

**منوی اصلی:**
- داشبورد
- مدیریت وظایف (فقط مدیر)
- وظایف من (فقط کارمند)
- پیام‌ها (فقط کارمند)
- تحلیل‌ها (فقط مدیر)

**میانبرها:**
- سفارشات
- راهنما
- تنظیمات

**بخش پایین:**
- پروفایل کاربر با آواتار
- نمایش وضعیت آنلاین
- دکمه خروج
- انتخاب نقش (برای تست)

### 3. نویگیشن بار پایین
- **حذف شد** - تمام صفحات از طریق ساید بار قابل دسترسی هستند
- فضای بیشتری برای محتوای اصلی

### 4. دکمه سفارش جدید
- قرار گرفته در پایین بخش "عملیات سریع" در داشبورد
- دکمه تمام عرض (Full Width) با گرادینت سبز
- متن: "ایجاد سفارش جدید +"
- فقط برای مدیر نمایش داده می‌شود
- با کلیک مودال ایجاد سفارش باز می‌شود

### 5. گراند کلی
- گرادینت آبی نیلی زیبا
- انیمیشن نرم گرادینت
- افکت Glass Morphism برای کارت‌ها

## فایل‌های تغییر یافته

### index.html
- جایگزینی هدر با ساید بار
- حذف تب‌های بالای صفحه
- اضافه کردن دکمه همبرگر برای موبایل
- تغییر نویگیشن بار پایین

### styles.css
- استایل‌های ساید بار
- انیمیشن‌های گرادینت
- افکت‌های Glass Morphism
- اسکرول بار سفارشی

### js/app.js
- اضافه کردن متغیر `sidebarOpen`
- تابع `getPageTitle()` برای عنوان صفحات
- مدیریت وضعیت ساید بار

## نحوه استفاده

### باز کردن ساید بار در موبایل
```javascript
sidebarOpen = true
```

### بستن ساید بار
```javascript
sidebarOpen = false
```

### تغییر صفحه
```javascript
currentPage = 'dashboard'  // یا 'tasks', 'orders', 'profile', ...
```

## ویژگی‌های Responsive

- **موبایل (< 1024px):** ساید بار به صورت overlay نمایش داده می‌شود
- **تبلت و دسکتاپ (≥ 1024px):** ساید بار همیشه باز است
- محتوای اصلی به صورت خودکار جابجا می‌شود

## رنگ‌ها

- **پس‌زمینه اصلی:** گرادینت آبی نیلی (blue-900 → indigo-900 → blue-800)
- **ساید بار:** گرادینت آبی (blue-600 → blue-700 → blue-800)
- **دکمه فعال:** سفید با شفافیت 20%
- **دکمه جدید:** سبز (green-500 → emerald-600)
- **نویگیشن پایین:** آبی تیره با افکت blur

## نکات مهم

1. ساید بار در دسکتاپ همیشه باز است و نمی‌توان آن را بست
2. در موبایل با کلیک روی overlay یا دکمه X بسته می‌شود
3. تمام صفحات از طریق ساید بار قابل دسترسی هستند
4. دکمه "ایجاد سفارش جدید" در پایین بخش عملیات سریع داشبورد قرار دارد
5. نویگیشن بار پایین حذف شده و فضای بیشتری برای محتوا وجود دارد

## مشکلات احتمالی و راه‌حل

### ساید بار نمایش داده نمی‌شود
- مطمئن شوید Alpine.js بارگذاری شده است
- متغیر `sidebarOpen` را چک کنید

### محتوا زیر ساید بار قرار می‌گیرد
- کلاس `lg:mr-80` به تگ `<main>` اضافه شده است
- در موبایل این margin حذف می‌شود
- padding پایین از `pb-24` به حالت عادی تغییر کرده (نویگیشن پایین حذف شد)

### انیمیشن‌ها کار نمی‌کنند
- Tailwind CSS و Alpine.js را چک کنید
- فایل `styles.css` را بررسی کنید
