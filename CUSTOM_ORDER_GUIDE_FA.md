# راهنمای سیستم سفارش سفارشی

## خلاصه تغییرات

این سند تغییرات اعمال شده برای سیستم سفارش سفارشی را شرح می‌دهد.

## 1. تغییرات فرم ایجاد سفارش (کارمند)

### قبل از تغییرات:
- فیلد قیمت (عددی)
- فیلد واحد پول (انتخابی: تومان، دلار، دینار)
- فیلد تاریخ تحویل (متنی)

### بعد از تغییرات:
- ✅ **حذف شد**: فیلد قیمت
- ✅ **حذف شد**: فیلد واحد پول
- ✅ **اضافه شد**: فیلد ساعت تحویل (time input)
- ✅ **بهبود یافت**: تاریخ و ساعت در کنار هم قرار گرفتند

### مسیر فایل:
```
js/employee.js
```

### تابع مربوطه:
```javascript
employeeModule.showCreateCustomOrderModal()
```

## 2. ساختار داده سفارش

```javascript
{
```

### فیلدهای حذف شده:

- ❌ `workPrices`

## 3. صفحه وظایف عامل

### ویژگی‌های جدید:

#### 3.1. نمایش کارت‌ها به صورت استک (Stack)
- کارت‌ها به ترتیب مهلت تحویل مرتب می‌شوند
- هر کارت کمی پایین‌تر از کارت قبلی قرار می‌گیرد
- افکت سایه برای ایجاد حس عمق

```css
transform: translateY(${index * 2}px);
box-shadow: 0 ${4 + index * 2}px ${8 + index * 4}px rgba(0,0,0,0.3);
```

#### 3.2. تایمر شمارش معکوس
- نمایش زمان باقی‌مانده به صورت زنده
- فرمت: "X روز و Y ساعت" یا "X ساعت و Y دقیقه" یا "X دقیقه و Y ثانیه"
- به‌روزرسانی هر ثانیه

```javascript
window.updateOrderTimer(orderId, deadlineDateTime)
```

#### 3.3. هشدار سفارشات منقضی شده
- حاشیه قرمز در سمت راست کارت
- انیمیشن pulse برای جلب توجه
- تغییر رنگ تایمر به قرمز
- پیام "زمان تحویل گذشته!"

```css
.border-red-500 {
    animation: pulse-red 2s infinite;
}
```

#### 3.4. هشدار سفارشات فوری (کمتر از 1 ساعت)
- تایمر زرد رنگ
- انیمیشن چشمک‌زن
- آیکون ساعت زرد

```javascript
const isUrgent = timeRemaining && timeRemaining < 3600000;
```

4. سفارشات منقضی شده با حاشیه قرمز مشخص می‌شوند
5. سفارشات فوری (کمتر از 1 ساعت) با رنگ زرد نمایش داده می‌شوند

## 9. مشکلات احتمالی و راه‌حل

### مشکل: تایمرها کار نمی‌کنند
**راه‌حل:**
- مطمئن شوید `startAgentTimers()` فراخوانی شده است
- کنسول مرورگر را بررسی کنید
- مطمئن شوید `deadlineDateTime` به درستی ذخیره شده است

### مشکل: حاشیه قرمز نمایش داده نمی‌شود
**راه‌حل:**
- فایل `css/agent-tasks.css` را بررسی کنید
- مطمئن شوید CSS به `index.html` لینک شده است
- کش مرورگر را پاک کنید

### مشکل: کارت‌ها به درستی مرتب نمی‌شوند
**راه‌حل:**
- تابع sort را بررسی کنید:
```javascript
myOrders.sort((a, b) => {
    if (!a.deadlineDateTime) return 1;
    if (!b.deadlineDateTime) return -1;
    return new Date(a.deadlineDateTime) - new Date(b.deadlineDateTime);
});
```
