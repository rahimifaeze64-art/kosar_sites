تست سیستم مدیریت تحصیلی

اطلاعات پروژه

  • نام: سیستم مدیریت تحصیلی دانشجویان بین‌المللی
  • نوع: Progressive Web App (PWA)
  • معماری: ماژولار و قابل تبدیل به Flutter
  • زبان: فارسی/عربی (RTL)
  • فریمورک: Alpine.js + Tailwind CSS

نحوه تست

شروع: سیستم با نقش مدیر شروع می‌شود
تغییر نقش: از منوی بالای صفحه نقش را تغییر دهید
تست: هر نقش را جداگانه تست کنید
PWA: از فوتر نویگیشن استفاده کنید

مراحل تست

  1. سیستم با نقش مدیر شروع می‌شود
  2. از منوی بالای صفحه نقش را تغییر دهید
  3. هر نقش را جداگانه تست کنید و قابلیت‌هایش را بررسی کنید
  4. از فوتر نویگیشن (PWA) برای جابه‌جایی استفاده کنید
  5. سفارش جدید ایجاد کنید و فرآیند کامل را تست کنید
شروع تست سیستم

نکات مهم:

  • • سیستم بدون لاگین شروع می‌شود
  • • تمام داده‌ها در localStorage ذخیره می‌شود
  • • سیستم PWA است و قابل نصب روی موبایل
  • • برای تست کامل، نقش‌های مختلف را امتحان کنید
  • • این یک پروتوتایپ است و برای استفاده واقعی نیاز به backend دارد
