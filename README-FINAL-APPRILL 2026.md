## 🚀 راه‌اندازی سریع (Quick Start)

### مرحله 1: راه‌اندازی بک‌اند

```bash
# ورود به پوشه بک‌اند
cd backend

# نصب وابستگی‌ها (اگر نصب نشده)
pip install -r requirements.txt

# اجرای migrations (اگر نیاز است)
python manage.py migrate

# راه‌اندازی سرور
python manage.py runserver
```

✅ بک‌اند روی `http://127.0.0.1:8000` اجرا می‌شود

### مرحله 2: راه‌اندازی فرانت‌اند

```bash
# بازگشت به پوشه اصلی
cd ..

# راه‌اندازی وب سرور ساده
python -m http.server 8080
```

✅ فرانت روی `http://localhost:8080` اجرا می‌شود

### مرحله 3: تست اتصال

مرورگر را باز کنید و به یکی از این آدرس‌ها بروید:

- **صفحه تست:** http://localhost:8080/test-connection.html
- **صفحه اصلی:** http://localhost:8080/index.html
- **پنل ادمین:** http://127.0.0.1:8000/admin/

## 🐛 رفع مشکلات رایج

### مشکل 1: CORS Error
**علامت:** خطای CORS در Console

**راه‌حل:**
```python
# backend/config/settings.py
CORS_ALLOWED_ORIGINS = [
    "http://localhost:8080",
    "http://127.0.0.1:8080",
]
```

### مشکل 2: Static Files Warning
**علامت:** Warning درباره پوشه static

**راه‌حل:**
```bash
cd backend
mkdir static
```

### مشکل 3: فایل‌های لوکال لود نمی‌شوند
**علامت:** 404 برای فایل‌های assets

**راه‌حل:**
- مطمئن شوید وب سرور در پوشه اصلی پروژه اجرا شده
- مسیرهای فایل‌ها را چک کنید

### مشکل 4: API پاسخ نمی‌دهد
**علامت:** خطای Network در Console

**راه‌حل:**
- مطمئن شوید بک‌اند روی پورت 8000 اجراست
- `enableAPI = true` در `js/api.js` باشد
