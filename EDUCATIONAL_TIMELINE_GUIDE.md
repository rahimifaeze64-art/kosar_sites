# راهنمای Timeline مراحل تحصیلی

## خلاصه تغییرات

یک **Timeline سه‌ستونی تعاملی** برای مراحل تحصیلی دانشجویان به صفحه ویرایش پروفایل اضافه شد.

## ویژگی‌های اصلی

### 1. نمایش Timeline
- Timeline سه‌ستونی با 22 مرحله تحصیلی
- نمایش در **بالای صفحه ویرایش پروفایل** (قبل از کارت اطلاعات شخصی)
- طراحی مدرن با انیمیشن‌های روان
- نوار پیشرفت دینامیک
- **دایره‌های بزرگ** (80x80 پیکسل - دو برابر اندازه قبلی)
- **نمایش سه‌ستونی** برای مشاهده آسان همه مراحل بدون نیاز به اسکرول

### 2. مراحل تحصیلی
مراحل به ترتیب:
1. محضر و اصالت
2. تنزیل نمره
3. تعدیل
4. ایرانداک خطه
5. ایرانداک رساله
6. مدرک لغت
7. ایجاد کردش
8. حاتمی
9. بارگزاری لغت
10. آزفا
11. ترجمه به اسماعیلی
12. دادگر
13. ارسال کد به تهران
14. وثیقه
15. تصدیق
16. تنضید
17. استلال
18. تجلید
19. ختم تجلید
20. قطعی
21. مدارک سابقه
22. ارسال

### 3. قابلیت‌های تعاملی
- ✅ **تیک زدن**: کلیک روی دایره برای تکمیل مرحله
- ❌ **برداشتن تیک**: کلیک مجدد روی دایره برای لغو تکمیل
- 📝 **یادداشت**: کلیک روی عنوان مرحله برای باز کردن صفحه یادداشت
- 📅 **ثبت تاریخ**: تاریخ تکمیل به صورت خودکار ذخیره می‌شود
- 📊 **نوار پیشرفت**: به‌روزرسانی خودکار درصد پیشرفت
- 💾 **ذخیره خودکار**: تغییرات بلافاصله در localStorage ذخیره می‌شوند
- 📋 **یادداشت اختیاری**: امکان افزودن توضیحات برای هر مرحله

### 4. طراحی بصری
- دایره‌های بزرگ (80x80 پیکسل) برای هر مرحله
- چیدمان سه‌ستونی برای نمایش بهتر
- رنگ سبز برای مراحل تکمیل شده
- رنگ خاکستری برای مراحل در انتظار
- نمایش آیکون یادداشت برای مراحل دارای توضیحات
- انیمیشن hover و scale
- Modal زیبا برای یادداشت‌ها

## فایل‌های تغییر یافته

### 1. `js/employee.js`
```javascript
// متدهای اضافه شده:
- getEducationalStepsTimeline(student) // نمایش Timeline سه‌ستونی
- getDefaultEducationalSteps() // مراحل پیش‌فرض با فیلد notes
- toggleEducationalStep(studentId, stepIndex) // تیک زدن/برداشتن
- showStepDetailsModal(studentId, stepIndex) // نمایش modal یادداشت
- saveStepNotes(studentId, stepIndex) // ذخیره یادداشت
```

### 2. `js/employee-student-edit.js`
```javascript
// تغییرات:

- ذخیره educationalSteps در saveStudentProfile
```

## نحوه استفاده

### برای کاربران:
1. وارد صفحه "دانشجویان" شوید
2. روی دکمه "ویرایش پروفایل" کلیک کنید
3. Timeline مراحل تحصیلی در بالای صفحه نمایش داده می‌شود
4. **روی دایره کلیک کنید** تا مرحله تیک بخورد
5. **روی عنوان مرحله کلیک کنید** تا صفحه یادداشت باز شود
6. یادداشت خود را وارد کنید و ذخیره کنید
7. تغییرات به صورت خودکار ذخیره می‌شوند

### برای توسعه‌دهندگان:
```javascript
// دریافت مراحل پیش‌فرض
const steps = employeeModule.getDefaultEducationalSteps();

// تغییر وضعیت یک مرحله
employeeModule.toggleEducationalStep(studentId, stepIndex);

// نمایش modal یادداشت
employeeModule.showStepDetailsModal(studentId, stepIndex);

// ذخیره یادداشت
employeeModule.saveStepNotes(studentId, stepIndex);

// دریافت HTML Timeline
const timelineHTML = employeeModule.getEducationalStepsTimeline(student);
```

## نکات مهم

1. **ذخیره‌سازی**: تمام تغییرات در `localStorage` با کلید `students_data` ذخیره می‌شوند
2. **مقداردهی اولیه**: اگر دانشجو `educationalSteps` نداشته باشد، به صورت خودکار مقداردهی می‌شود
3. **تاریخ فارسی**: تاریخ تکمیل به صورت فارسی ذخیره می‌شود
4. **چیدمان سه‌ستونی**: مراحل به 3 ستون تقسیم می‌شوند (هر ستون حدود 7-8 مرحله)
5. **دایره‌های بزرگ**: اندازه دایره‌ها 80x80 پیکسل است
6. **دو نوع کلیک**: 
   - کلیک روی دایره = تیک زدن/برداشتن
   - کلیک روی عنوان = باز کردن modal یادداشت
7. **یادداشت اختیاری**: می‌توانید برای هر مرحله توضیحات اضافه کنید

## مشکلات احتمالی و راه‌حل

### مشکل: Timeline نمایش داده نمی‌شود
**راه‌حل**: 
- مطمئن شوید `employee.js` قبل از `employee-student-edit.js` لود شده است
- کنسول مرورگر را برای خطاها بررسی کنید

### مشکل: تیک زدن کار نمی‌کند
**راه‌حل**:
- مطمئن شوید `UTILS.showNotification` تعریف شده است
- localStorage را پاک کنید و دوباره امتحان کنید

### مشکل: تغییرات ذخیره نمی‌شوند
**راه‌حل**:
- مطمئن شوید دکمه "ذخیره تغییرات" کلیک شده است
- localStorage مرورگر را بررسی کنید
