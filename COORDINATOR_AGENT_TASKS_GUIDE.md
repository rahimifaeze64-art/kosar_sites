# راهنمای صفحه "وظایف عامل‌ها" برای کارمند

## نمای کلی

صفحه "وظایف عامل‌ها" به کارمند ها اجازه می‌دهد تا:
- تمام وظایفی که برای عامل‌ها تعریف کرده‌اند را مشاهده کنند
- وضعیت real-time وظایف را پیگیری کنند
- از تغییرات وضعیت توسط عامل‌ها مطلع شوند
- وظایف را مدیریت و حذف کنند

## دسترسی به صفحه

1. به عنوان کارمند وارد سیستم شوید
2. از منوی سمت راست، گزینه "وظایف عامل‌ها" را انتخاب کنید

## ویژگی‌های صفحه

### 1. آمار کلی (4 کارت)
- **کل وظایف**: تعداد کل وظایف تعریف شده
- **در انتظار**: وظایفی که عامل هنوز شروع نکرده
- **در حال انجام**: وظایفی که عامل در حال انجام است
- **تکمیل شده**: وظایفی که عامل تکمیل کرده

### 2. فیلترهای هوشمند
چهار دکمه فیلتر برای نمایش وظایف:
- **همه**: نمایش تمام وظایف
- **در انتظار**: فقط وظایف شروع نشده
- **در حال انجام**: فقط وظایف در دست اقدام
- **تکمیل شده**: فقط وظایف انجام شده

### 3. نمایش Grid (3 کارت در سطر)
- طراحی مشابه صفحه وظایف عامل
- Responsive برای تمام اندازه‌های صفحه
- انیمیشن hover برای کارت‌ها

### 4. اطلاعات هر کارت

**اطلاعات اصلی:**
- شماره وظیفه
- عنوان و توضیحات
- نام عامل تخصیص داده شده
- نام دانشجو
- وضعیت فعلی (با آیکون متحرک)
- برچسب فوری (برای اولویت بالا)

**تایمر شمارش معکوس:**
- 2 مربع: روز و ساعت باقی‌مانده
- به‌روزرسانی خودکار هر دقیقه
- هشدار قرمز برای وظایف نزدیک به مهلت
- پیام "مهلت گذشته است!" برای دیرکرد

**اعلان تغییر وضعیت:**
- نمایش پیام "عامل کار را شروع کرد" (آبی)
- نمایش پیام "عامل وظیفه را تکمیل کرد" (سبز)
- نمایش تاریخ و ساعت تغییر وضعیت

**فایل ضمیمه:**
- نمایش نام فایل
- دکمه دانلود

**دکمه‌های عملیات:**
- **جزئیات**: مشاهده اطلاعات کامل وظیفه
- **حذف**: حذف وظیفه (فقط برای وظایف تکمیل نشده)

### 5. مودال جزئیات وظیفه

با کلیک روی "جزئیات"، مودالی نمایش داده می‌شود شامل:
- عنوان و توضیحات کامل
- وضعیت، اولویت
- نام عامل و دانشجو
- مهلت تحویل و تاریخ ایجاد
- فایل ضمیمه با دکمه دانلود
- اطلاعات تکمیل (در صورت تکمیل شدن)

## نحوه استفاده

### مشاهده وظایف

1. **مشاهده همه وظایف:**
   - به صفحه "وظایف عامل‌ها" بروید
   - تمام وظایف تعریف شده نمایش داده می‌شود

2. **فیلتر کردن:**
   - روی یکی از دکمه‌های فیلتر کلیک کنید
   - فقط وظایف با وضعیت انتخابی نمایش داده می‌شود

3. **پیگیری زمان:**
   - تایمر هر وظیفه به صورت خودکار به‌روزرسانی می‌شود
   - وظایف نزدیک به مهلت با رنگ قرمز هشدار داده می‌شوند

### مدیریت وظایف

1. **مشاهده جزئیات:**
   - روی دکمه "جزئیات" کلیک کنید
   - اطلاعات کامل وظیفه نمایش داده می‌شود

2. **دانلود فایل:**
   - روی آیکون دانلود کلیک کنید
   - فایل ضمیمه دانلود می‌شود

3. **حذف وظیفه:**
   - روی دکمه حذف (سطل زباله) کلیک کنید
   - تایید حذف را بدهید
   - وظیفه حذف می‌شود

4. **تعریف وظیفه جدید:**
   - روی دکمه "تعریف وظیفه جدید" کلیک کنید
   - فرم تعریف وظیفه باز می‌شود

## اعلان‌های Real-time

### وقتی عامل کار را شروع می‌کند:
```
✓ وضعیت از "در انتظار" به "در حال انجام" تغییر می‌کند
✓ پیام آبی رنگ نمایش داده می‌شود:
  "عامل کار را شروع کرد - تاریخ و ساعت"
✓ آیکون وضعیت به spinner متحرک تغییر می‌کند
```

### وقتی عامل وظیفه را تکمیل می‌کند:
```
✓ وضعیت از "در حال انجام" به "تکمیل شده" تغییر می‌کند
✓ پیام سبز رنگ نمایش داده می‌شود:
  "عامل وظیفه را تکمیل کرد - تاریخ و ساعت"
✓ تایمر حذف می‌شود
✓ باکس سبز "وظیفه تکمیل شده" نمایش داده می‌شود
✓ دکمه حذف غیرفعال می‌شود
```

## نکات مهم

⚠️ **به‌روزرسانی خودکار**: تغییرات وضعیت توسط عامل بلافاصله در صفحه کارمند نمایش داده می‌شود (با refresh صفحه)

⚠️ **حذف وظیفه**: فقط وظایف تکمیل نشده قابل حذف هستند

⚠️ **تایمر**: تایمرها فقط برای وظایف تکمیل نشده نمایش داده می‌شوند

⚠️ **فیلتر**: فیلترها تعداد وظایف را در کنار هر دکمه نمایش می‌دهند

## عیب‌یابی

### وظایف نمایش داده نمی‌شوند:

1. **بررسی employeeId:**
   ```javascript
   const currentUser = JSON.parse(localStorage.getItem('currentUser'));
   console.log('employee ID:', currentUser.id);
   ```

2. **بررسی وظایف:**
   ```javascript
   const tasks = JSON.parse(localStorage.getItem('agent_tasks') || '[]');
   console.log('All tasks:', tasks);
   console.log('My tasks:', tasks.filter(t => t.employeeId === currentUser.id));
   ```

### تایمر کار نمی‌کند:

```javascript
// در Console:
employeeModule.startemployeeTimers();
```

### فیلتر کار نمی‌کند:

```javascript
// Refresh صفحه:
location.reload();
```

## مثال سناریو

### سناریو کامل:

1. **کارمند (زهرا):**
   - وظیفه‌ای برای عامل معصومی تعریف می‌کند
   - به صفحه "وظایف عامل‌ها" می‌رود
   - وظیفه را با وضعیت "در انتظار" می‌بیند

2. **عامل (عامل معصومی):**
   - به صفحه "وظایف من" می‌رود
   - وظیفه را می‌بیند
   - روی "شروع کار" کلیک می‌کند

3. **کارمند (زهرا):**
   - صفحه را refresh می‌کند
   - وضعیت به "در حال انجام" تغییر کرده
   - پیام آبی: "عامل کار را شروع کرد" می‌بیند

4. **عامل (عامل معصومی):**
   - کار را انجام می‌دهد
   - روی "تکمیل" کلیک می‌کند

5. **کارمند (زهرا):**
   - صفحه را refresh می‌کند
   - وضعیت به "تکمیل شده" تغییر کرده
   - پیام سبز: "عامل وظیفه را تکمیل کرد" می‌بیند
   - باکس سبز "وظیفه تکمیل شده" را می‌بیند

## خلاصه

صفحه "وظایف عامل‌ها" یک ابزار قدرتمند برای کارمند ها است که به آن‌ها اجازه می‌دهد:

✅ تمام وظایف تعریف شده را در یک نگاه ببینند
✅ وضعیت real-time وظایف را پیگیری کنند
✅ از پیشرفت کار عامل‌ها مطلع شوند
✅ وظایف را مدیریت و حذف کنند
✅ زمان باقی‌مانده تا مهلت را ببینند
✅ وظایف را بر اساس وضعیت فیلتر کنند
